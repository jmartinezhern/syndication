# Syndication - A simple news aggregation server

## Features
* JSON REST API
* Unix socket based Administration API
* Let's Encrypt through Echo framework (experimental)
* Support for SQLite, MySQL and Postgres

## Building

```
$ dep ensure
$ go build
```
